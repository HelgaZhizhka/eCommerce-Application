# Отчёт: агентный рефакторинг eCommerce-приложения

> Материал для поста. Все цифры — из git (даты коммитов, `git diff --shortstat`),
> не оценки задним числом. Календарный период: **10–20 июня 2026** (part-time
> с перерывами). Рефакторинг (фазы 0–6) завершён на `develop`.

## Что это было

Учебный проект RS School 2023 года (витрина на Commercetools) переведён с
мёртвого стека на современный. Работали **AI-агенты (Claude Code)** под ревью
человека: агент пишет код и сам себя проверяет, человек принимает решения и
смотрит результат глазами.

**Модели:**
- **Fable 5** — анализ кодовой базы, план рефакторинга, фазы 0–3.
- **Opus 4.8** — фазы 4–6 (формы, UI, полный снос MUI, тесты/покрытие/CI/a11y,
  полировка тёмной темы), код-ревью.

## Стек: было → стало

| | Было (2023) | Стало (2026) |
|---|---|---|
| Сборка | Create React App (мёртв) | **Vite 7** |
| UI-runtime | React 18 | **React 19** |
| Язык | TypeScript 4.9 | **TypeScript 5.9** strict |
| Состояние | MobX (4 стора, ~1050 строк) | **TanStack Query 5** + **Zustand 5** |
| Формы | Formik + Yup + самописная машина | **react-hook-form + zod 4** |
| SDK | `sdk-client-v2` (deprecated) | **`@commercetools/ts-client` 4** + BFF |
| Auth | clientSecret в браузерном бандле | **BFF на Netlify Functions**, секрет на сервере |
| Тесты | Jest (CRA), ~0% логики | **Vitest 4 + MSW 2 + Playwright** (98 unit/интеграционных + 11 e2e в CI, пороги покрытия 80%) |
| Линт | ESLint 8 + airbnb (мёртв) | **ESLint 9 flat config** |
| UI / стили | SCSS-модули + MUI + sx (3 системы) | **Tailwind 4 + Radix + lucide** (MUI/emotion удалены, legacy SCSS вычищен полностью — одна система стилей) |

## Объём работы (из git)

Смержено в `develop` (PR #216–#223):
- **49 коммитов** (без merge) + **8 PR** (#216–#223)
- **349 файлов** изменено, **+29 900 / −9 897** строк
- Удалено целиком: 4 MobX-стора, самописная валидация (~1100 строк),
  **MUI + emotion (5 пакетов)** и 9 других мёртвых зависимостей, 100+
  закоммиченных артефактов сборки, дубли компонентов (`HeaderMobile`,
  `FilterMobile`, `SortMobile`)

Плюс **часть B смержена** (PR #225, чистка SCSS): ещё ~20 коммитов, **89 файлов**,
**+1 041 / −2 364**. Все `*.module.scss` переведены на Tailwind, глобальные partials
свёрнуты в один `tailwind.css` — **0 `.scss` в дереве**, `sass` и `classnames` удалены.

**Фаза 6 смержена** (PR #227–#241): **15 коммитов**, **33 файла**, **+2 348 / −262**.
MSW 2 + интеграционные тесты (каталог/фильтры, корзина, auth, профиль) — всего
**98 unit/интеграционных** тестов (+ 11 e2e), пороги покрытия 80% на `services/*`
+ `queries/*` (в CI). Документация (README + 3 ADR), e2e против Netlify-превью в
CI, a11y-проход и полировка тёмной темы (см. ниже).

## Главные результаты

- 🔒 **Безопасность:** `clientSecret` убран из бандла (был публичен с 2023).
  Проверено grep'ом по прод-сборке → 0 вхождений.
- 🧹 **MobX / Formik / CRA / MUI удалены** из зависимостей полностью.
- 🎨 **UI переведён на Tailwind 4** (engine-swap, look 1:1): иконки → lucide,
  интерактив (Slider/Select/Dialog) → Radix, общие примитивы Button/Select/Dialog
  + RHF-адаптеры. **0 импортов `@mui`** — бандл больше не тащит MUI/emotion.
- 🔗 **Состояние каталога — в URL** (фильтры/сортировка/страница шарятся ссылкой).
- 🐛 **Починены баги**, найденные по ходу: гонка версий корзины, гонка
  `setTimeout` при регистрации, несовпадение email-валидации, индекс по стране,
  невалидная страна «EU», все адреса помечались default, опечатка «Sing up».
  Визуальное ревью поймало регрессию `CssBaseline` (фон/шрифт body) после сноса MUI.
- 🤖 **Код-ревью самими агентами:** ревью PR #221 запущено отдельным субагентом
  (свежий контекст) → «Approve with nits», 3 нита пофикшены отдельным PR #222.
- ✅ **Воспроизводимость:** lockfile + CI (GitHub Actions) на каждый PR; гейт
  (build + tsc + eslint + unit + e2e) прогонялся перед каждым «готово».
- 🧪 **Фаза 6 — качество:** MSW 2 + **98 unit/интеграционных** тестов (каталог/
  фильтры, корзина, auth, профиль), пороги покрытия **80%** на API-слое в CI,
  Playwright против Netlify-превью, a11y-проход (label/aria, focus-trap),
  README + 3 ADR.
- 🌗 **Полировка тёмной темы:** живое ревью вскрыло семейство багов из-за
  выключенного Tailwind preflight (поля/кнопки/иконки и `text-content`/`bg-*`
  тащили светлые цвета в тёмной теме — Tailwind v4 «запёк» семантические
  `@theme`-токены). Починено в корне; свет 1:1 не тронут, по всем страницам
  0 «тёмного-на-тёмном».
- 🧭 **Фикс навигации:** страницы «рендерились снизу» при переходах —
  `useLayoutEffect` + `AnimatePresence mode="wait"`.

## Как обеспечивалось качество

- **Harness** в репозитории: `CLAUDE.md` (контракт), `docs/refactor/PROGRESS.md`
  (статус между сессиями), чек-листы фаз, `scripts/verify.sh` (единые ворота).
- **Сетка до миграции:** сначала 11 Playwright-сценариев на старом поведении,
  потом замена слоёв — каждое падение ловилось до коммита.
- **Гейт перед каждым «готово»:** build + typecheck + ESLint + 41 unit + 11 e2e.
  Прогонялся после каждой группы; e2e дважды подряд (flake-check).
- **Фаза за фазой, PR за PR** — поведение приложения заморожено, ничего не
  ломалось между шагами; UI-фаза дополнительно режется на под-PR по группам
  с визуальным ревью в живом превью (свет/тёмная, desktop/mobile).

**Рефакторинг завершён — все 7 фаз (0–6) смержены в `develop`** (PR #216–#241).

- Фаза 5: **MUI и legacy SCSS удалены полностью** — одна система стилей

  итогам живого ревью.

  заморожены (1:1), ревью человеком в живом превью (свет/тёмная, desktop/mobile).

по-прежнему отдаёт сборку 2023 года.
